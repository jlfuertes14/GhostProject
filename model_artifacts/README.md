# Model Artifacts

This directory stores the trained ML model files:

- `delay_model.pkl` — XGBoost classifier for project delay prediction
- `budget_model.pkl` — XGBoost regressor for budget estimation
- `preprocessor.pkl` — Fitted encoder/scaler pipeline

These are generated by the Google Colab notebooks and consumed by the FastAPI backend.

> **Note**: `.pkl` files are excluded from git via `.gitignore` due to their size.
> Re-generate them by running the training notebooks.
